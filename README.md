Activity 1: API Integration Testing using
pytest-bdd

@ Objective

In this activity, you will:

. Practice API Integration Testing using pytest-bdd
. Work with a predefined BDD Feature File
. Implement step definitions to make all scenarios pass
. Validate both happy path and negative scenarios

You will be testing the public API:

Base URL:
https://api.restful-api.dev/objects

Important Instructions

. The BDD Feature File is provided below.
. Your task is to implement the step definitions and any supporting code required to
make all scenarios pass.
. You are NOT allowed to modify the feature file.
. All assertions must be placed inside Then steps only.
. Focus on integration flow and error handling.
